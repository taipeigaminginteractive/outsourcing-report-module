# Contributing Guidelines

This repository is used for **authorized, task-based outsourced development only**.
It is not an open-source collaboration project.

本 Repository 僅用於 **經授權之任務型委外開發**，  
並非開源協作專案。

## Contribution Rules

- Fork-based contributions only
- One assigned task per pull request
- One pull request per Issue

- 僅接受以 Fork 方式進行的貢獻
- 每個 Pull Request 僅對應一個已指派的任務
- 每個 Issue 僅接受一個對應的 Pull Request


## Authorization Requirement

This repository does **NOT** accept unsolicited contributions.

Only contributors **explicitly authorized by the repository owner**
and assigned via a GitHub Issue may submit pull requests.

Pull requests submitted without prior authorization or assignment
will be closed without review.

本 Repository **不接受任何未經授權的貢獻**。

僅限於 **經 Repository Owner 明確授權**，  
且已透過 GitHub Issue 指派之貢獻者，  
方可提交 Pull Request。

任何未經事前授權或未有 Issue 指派的 Pull Request，  
將在不另行說明的情況下關閉，且不進行審查。

## Review Policy

Acceptance is based solely on task completion and requirements
是否接受僅依任務完成度與需求符合度判定
